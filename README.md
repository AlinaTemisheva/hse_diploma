# 📍 Цифровой продукт для онбординга внешних преподавателей в университете
MVP цифрового продукта для онбординга приглашённых преподавателей из бизнес-среды. На основе интервью и анализа документов выявлена фрагментарность процессов адаптации, вследствие чего предложено решение в виде единой цифровой среды с микрообучением и on-demand доступом к информации. Проверены гипотезы, разработаны UX/UI, прототип и микрокурс. Доступно администрирование сервиса (создание курсов, добавление преподавателей, проставление задач, документов и управление FAQ).
